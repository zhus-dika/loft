# Loft System
###  To deploy page:
```sh
$ git clone https://github.com/zhus-dika/loft.git
$ cd loft
```
##Scripts of package.json:

| Script | Target |
| ------ | ------ |
| start | Run web app on localhost:5000

#### To evaluate script:
```sh
$ npm run script_name
```

##### Or:
```sh
$ yarn script_name
```
